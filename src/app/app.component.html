The classic puzzle game in which mines are randomly distributed on the map, and the user clicks squares to explore.
Explore an empty square to learn the number of mines adjacent, but if you explore a mined square it's game over!
Right click an unexplored square to mark it as believed to be mined.
Right click an explored square to explore all unmarked squares around it.
